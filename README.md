### Вычислитель отличий - определяет разницу между двумя структурами файлов

* Поддерживает yaml и json
* Отчеты в формате plain, stylish и json

### Установка

```bash
git clone https://github.com/GreedVal/php-project-48.git

make install

make validate
```
